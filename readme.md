Инструкция по пользованию:
1) Установите маску расширений файлов, которые будут обработаны программой, правильная маска выглядит так (без кавычек): ".txt .bin .exe".
2) Выберите папку, в которой программа будет искать файлы для обработки. Будут обработаны все доступные в папке файлы, расширения которых соответствуют маске. По умолчанию установлена папка с исполняемым файлом.
3) Выберите тип сохранения:
  а) При обработке исходный файл будет перезаписан.
  б) При создании нового файла будет создан файл формата "исходноеИмя_output1" в папке назначения.
5) Выберите папку для сохранения файлов. По умолчанию это папка с исполняемым файлом.
6) Установите нужные флажки.
7) Введите число-модификатор, с которым будет проведена операция xor. Число ограничено по величине программно, оно автоматически сбросится, если значение недопустимо.
8) Нажмите "Старт" и ожидайте завершения работы программы. В окне статуса отобразится сообщение об окончании работы программы.
9) 
Бизнес-логика работает в отдельном потоке, поэтому программа не перестанет отвечать, однако интерфейсом программы во время её работы пользоваться не получится в целях сохранения состояния параметров на время исполнения.

Кнопка "Стоп" экстренно прервет выполнение операции в рабочем потоке. Прерывание полностью безопасно.
Срабатывание по таймеру будет запускать сканирование и обработку файлов каждые Х секунд, соответствуя установленному периоду.


Компиляция:
```sh 
qmake xor_ported.pro
```
